\section{Javascript}
JavaScript è un linguaggio di programmazione multi paradigma orientato agli eventi, utilizzato sia nella programmazione lato client web che lato server (\ccgloss{NodeJs}) per la creazione di REST \ccgloss{API}, applicazioni desktop e embedded, siti web e applicazioni web e di effetti dinamici interattivi.

\section{JetBrains}
JetBrains (precedentemente IntelliJ) è un'azienda di sviluppo software i cui strumenti hanno come mercato gli sviluppatori software e i project manager. L'azienda offre una vasta gamma di ambienti di sviluppo integrati (IDE) per i linguaggi di programmazione e strumenti di revisione codice come \ccgloss{YouTrack}.
